# RHWP Mac Feature Status

기준 날짜: 2026-04-09

## 상태 기준

- `구현됨`: UI와 기본 동작이 연결되어 실제로 사용할 수 있는 상태
- `부분 구현`: UI 또는 엔진 경로 일부만 연결되어 있고, 세부 편집이나 완성도 보강이 더 필요한 상태
- `미구현`: 아직 placeholder 이거나 동작을 연결하지 않은 상태
- `확인 필요`: 현재 동작하지만 안정성이나 완성도를 계속 점검해야 하는 영역

## 구현됨

### 문서/세션

- HWP/HWPX 문서 열기
- 새 문서 생성
- HWP 저장
- 페이지 렌더 트리 기반 페이지 표시
- 줌, 스크롤, 현재 페이지 추적
- macOS 앱 번들 실행 경로
- MCP 기반 문서 열기, 읽기, 수정, 렌더, 저장

### 기본 편집

- 클릭 이동
- 방향키 이동
- 드래그 선택
- `cmd+a` 문서 전체 선택
- 기본 텍스트 입력
- 백스페이스
- 엔터
- 잘라내기, 복사, 붙여넣기
- snapshot 기반 undo/redo
- 찾기, 찾아 바꾸기

### 서식

- 글꼴 변경
- 글자 크기 변경
- 굵게, 기울임, 밑줄, 취소선
- 글자색, 형광펜
- 문단 정렬
- 줄 간격
- 글머리표
- 문단 번호
- 문단 수준 증가/감소
- 글자 모양 대화상자
- 문단 모양 대화상자

### 표

- 표 삽입
- 행/열 추가
- 행/열 삭제
- 셀 합치기
- 셀 나누기
- 표/셀 속성 대화상자

### 쪽/구역

- 쪽 나누기
- 단 나누기
- 편집 용지 대화상자
- 현재 구역 편집 용지 적용
- 모든 구역 편집 용지 일괄 적용

### 부가 기능

- 책갈피 목록, 추가, 이름 변경, 삭제, 이동
- 필드 목록과 값 편집
- 각주 삽입

## 부분 구현

### 머리말/꼬리말

- 빠른 템플릿 적용
- 현재 상태 조회
- 적용 범위별 존재 여부 조회
- 삭제
- 관리 대화상자

아직 부족한 점:

- 머리말/꼬리말 본문 텍스트 직접 편집
- 페이지 단위 이전/다음 이동
- 필드 삽입 세부 편집

### 선택/입력

- 문서 전체 선택은 여러 섹션까지 선택 rect가 보이도록 확장됨
- 빠른 입력 시 전체 문서 재로딩은 줄였음

아직 부족한 점:

- 빠른 타이핑에서 체감 지연 추가 최적화
- 한글 IME 조합 입력 안정성 추가 점검

### UI/UX

- 웹 버전의 `title_bar / menu_bar / style_bar` 구조를 참고한 macOS 전용 재구성
- 다수의 툴바/메뉴 항목이 실제 기능과 연결됨

아직 부족한 점:

- 웹 버전 대비 세부 interaction parity
- 일부 드롭다운/팝업의 마감 완성도

## 확인 필요

- 한글 IME 조합 입력 안정성

- `cmd+c / cmd+v / cmd+x / cmd+a` responder 체인 안정성
- 큰 문서에서 selection rect / render cache 성능

## 빌드와 실행

```bash
source "$HOME/.cargo/env"
./hwp-mac/scripts/build-rust.sh
export RHWP_LIB_SEARCH_PATH="$(pwd)/target/debug"
./hwp-mac/scripts/run-app.sh
```

## 빠른 개발 루프

진짜 hot reload는 아니지만, 소스 변경 시 다시 빌드하고 앱을 재실행하는 watch 스크립트를 쓸 수 있습니다.

```bash
source "$HOME/.cargo/env"
export RHWP_LIB_SEARCH_PATH="$(pwd)/target/debug"
./hwp-mac/scripts/dev-watch.sh
```
